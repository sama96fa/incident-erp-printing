# ERP — Impossible d’imprimer après mise à jour Windows (25 utilisateurs)

## Résumé
Depuis ce matin, plusieurs utilisateurs ne peuvent plus imprimer depuis l’ERP. Le problème est apparu après une mise à jour Windows.

## Description
- Symptôme : impression impossible depuis l’ERP (erreur ou impression bloquée)
- Périmètre : environ 25 utilisateurs
- Détection : 09h10

## Impact utilisateur
Blocage partiel du travail (documents ERP non imprimables) pour un groupe d’utilisateurs.

## Étapes de reproduction
1. Ouvrir l’ERP
2. Lancer l’impression d’un document (ex : bon, facture, état)
3. Constater l’échec d’impression ou le document bloqué en file
