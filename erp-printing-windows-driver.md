# ERP — Problème d’impression après mise à jour Windows (pilote incompatible)

## Prérequis
- Accès au poste Windows
- Droits pour réinstaller l’imprimante/pilote (ou assistance Support)
- Nom de l’imprimante (si connu)

## Étapes
1. Fermer l’ERP.
2. Redémarrer le poste (si pas déjà fait).
3. Vérifier si l’impression fonctionne depuis une autre application (test simple).
4. Supprimer puis réinstaller l’imprimante (si applicable).
5. Installer/forcer le **pilote compatible** (version validée).
6. Relancer l’ERP et tester une impression.

## Dépannage
- **Toujours impossible** : tester une autre imprimante (si disponible) pour isoler le problème.
- **Impression bloquée** : vider la file d’impression puis retester.
- **Après update Windows** : demander au Support de vérifier/forcer le pilote standard.

## Prévention
Après une mise à jour Windows, tester l’impression ERP sur un poste « pilote » avant déploiement large.
